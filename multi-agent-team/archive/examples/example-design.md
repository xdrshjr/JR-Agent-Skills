# Example: Design Task

## User Request

> "Design a landing page for my new meditation app called 'Mindful'. Target audience is busy professionals aged 25-40. Should feel calm, modern, and trustworthy. Include: hero section, features, testimonials, pricing, and CTA."

---

## PM Analysis

**Task Type:** UI/UX Design  
**Complexity:** Medium-High (multiple sections, brand identity)  
**Estimated Duration:** 20-25 minutes

---

## Team Configuration

| Role | Responsibilities |
|------|-----------------|
| **Brand Strategist** | Define visual identity, color palette, typography system |
| **UI Designer** | Create wireframes, design all sections, responsive layouts |
| **UX Copywriter** | Write compelling copy, micro-interactions, CTA buttons |
