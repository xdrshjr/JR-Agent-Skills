# Common Computer Science Conferences & Journals

This document provides information about major CS venues to help contextualize reviews.

## Top-Tier Machine Learning Conferences

### NeurIPS (Conference on Neural Information Processing Systems)
- **Focus**: Machine learning, neural networks, AI
- **Key Emphasis**: Theoretical rigor, reproducibility, code submission
- **Review Criteria**: Novelty, technical quality, experimental validation, clarity
- **Special Requirements**:
  - Code submission encouraged
  - Reproducibility checklist
  - Broader impact statement
- **Typical Paper Length**: 8 pages + unlimited references
- **Acceptance Rate**: ~20-25%

### ICML (International Conference on Machine Learning)
- **Focus**: Machine learning theory and applications
- **Key Emphasis**: Balance of theory and experiments, statistical significance
- **Review Criteria**: Technical soundness, significance, clarity
- **Special Requirements**:
  - Supplementary materials common
  - Statistical significance testing
- **Typical Paper Length**: 8 pages + unlimited references
- **Acceptance Rate**: ~20-25%

### ICLR (International Conference on Learning Representations)
- **Focus**: Deep learning, representation learning
- **Key Emphasis**: Open review process, reproducibility, code availability
- **Review Criteria**: Novelty, quality, clarity, significance
- **Special Requirements**:
  - Open review (reviews are public)
  - Code submission highly valued
  - Reproducibility important
- **Typical Paper Length**: No strict limit, typically 8-10 pages
- **Acceptance Rate**: ~25-30%

## Review Standards by Venue Type

### Conference Papers
- **Length**: Typically 8-10 pages
- **Scope**: Focused contribution
- **Experiments**: Sufficient to support claims
- **Timeline**: 3-4 months review
- **Revisions**: Usually no revision round (accept/reject)

### Journal Papers
- **Length**: 15-30 pages typical
- **Scope**: Comprehensive, complete work
- **Experiments**: Extensive, thorough evaluation
- **Timeline**: 6-12 months review
- **Revisions**: Multiple revision rounds common

### Workshop Papers
- **Length**: 4-6 pages typical
- **Scope**: Work in progress, preliminary results
- **Experiments**: Preliminary validation acceptable
- **Timeline**: 1-2 months review
- **Standards**: Lower bar than main conference

## Common Venue-Specific Requirements

### Reproducibility
- **High Priority**: NeurIPS, ICLR, ICML
- **Encouraged**: CVPR, ACL, EMNLP
- **Standard**: Most other venues

### Code Submission
- **Required/Strongly Encouraged**: ICLR, NeurIPS (reproducibility track)
- **Encouraged**: CVPR, ICML, ACL
- **Optional**: Most other venues

### Broader Impact Statement
- **Required**: NeurIPS
- **Encouraged**: ICLR, FAccT
- **Optional**: Most other venues

### Ethics Review
- **Formal Process**: ACL, EMNLP, NeurIPS
- **Considered**: Most major venues
- **Varies**: Smaller venues

## Adjusting Review Standards

When reviewing for different venues, adjust expectations:

### Top-Tier Conferences (NeurIPS, ICML, CVPR, ACL)
- **Novelty**: High bar, significant contribution expected
- **Quality**: Rigorous technical work
- **Experiments**: Comprehensive, strong baselines
- **Impact**: Clear significance to field

### Mid-Tier Conferences
- **Novelty**: Solid contribution, incremental acceptable
- **Quality**: Sound technical work
- **Experiments**: Adequate validation
- **Impact**: Useful to community

### Workshops
- **Novelty**: Interesting ideas, work in progress OK
- **Quality**: Preliminary work acceptable
- **Experiments**: Initial validation sufficient
- **Impact**: Potential for future work

### Journals
- **Novelty**: Substantial contribution
- **Quality**: Highly rigorous
- **Experiments**: Extensive, comprehensive
- **Impact**: Significant, lasting contribution

## Red Flags by Venue

### For Top-Tier Conferences
- Incremental improvements only
- Limited experimental validation
- Missing key baselines
- Poor writing quality
- Unclear contributions

### For Journals
- Insufficient depth or breadth
- Limited novelty
- Incomplete experimental evaluation
- Not significantly beyond conference version

### For Any Venue
- Technical errors
- Overclaiming
- Missing related work
- Unfair comparisons
- Ethical concerns not addressed
